# All Star Platform Vision

All star Platform is a complete development used to handle internal processes for All Star organization. There will be various applications maintaining business in an organized logic in order to optimize and perform different activities with different departments such as production or inventory. The main objetive for this development is to have a complete and modern client capable of managing daily activities that are currently handled using other methods.

This client will be specified to be used entirely for employees, because of that this project will be run only in internal environments where the security and performance are relevant aspects to have consider during the development. The general idea is to develop a powerful but lightweight application which solves operational problems within the company, the purpose is to have different modular applications for different user uses, almost as if it were an ERP.
